# Proyecto Integrador U2 - Walk Cycle en Blender

# Descripcion
Este proyecto consiste en la creación de una animación tipo "walk cycle" en Blender 2D.
Se realizó siguiendo una referencia, pero adaptando las poses de manera propia.

## Proceso

1. Se abrió Blender y se configuró el espacio de trabajo en 2D.
2. Se importó o creó el personaje (Kirby).
3. Se identificaron las poses principales del walk cycle:
   - Contacto
   - Paso
   - Elevación
4. Se dibujaron las poses clave.
5. Se añadieron los frames intermedios para suavizar la animación.
6. Se ajustó la velocidad de la animación.
7. Se exportó el resultado como video (.mp4).
